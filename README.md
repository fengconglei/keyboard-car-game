# 字母赛车键盘练习

一个面向儿童和键盘初学者的网页小游戏。玩家根据屏幕提示按下对应英文字母，小车会根据输入结果给出反馈，帮助用户在游戏过程中熟悉键盘字母位置、提升反应速度和键盘认知。

## 项目简介

本项目是一个纯前端小游戏，不依赖后端服务，也不需要安装复杂环境。打开 `index.html` 即可运行。

游戏界面包含：

- 顶部状态栏：展示得分、连击和剩余时间
- 中间赛车场景：显示目标字母、小车动画和操作提示
- 右侧控制面板：开始游戏、声音开关、速度调节
- 底部键盘地图：高亮当前需要按下的字母

## 功能特点

### 1. 字母识别练习

系统会随机生成 A-Z 中的目标字母，玩家需要在键盘上按下对应按键。

### 2. 屏幕键盘提示

页面底部会显示完整的字母键盘地图，并自动高亮当前目标字母，适合刚开始学习键盘位置的孩子使用。

### 3. 得分与连击机制

按对字母后会增加分数，并累计连击；连续答对可以获得更高分数。按错会扣少量分数，并清空当前连击。

### 4. 倒计时挑战

游戏有时间限制，玩家需要在限定时间内尽可能多地按对字母。

### 5. 难度调节

支持三种速度模式：

- 轻松：75 秒
- 标准：60 秒
- 挑战：45 秒

### 6. 声音反馈

游戏支持声音开关。开启后，系统会朗读当前目标字母，并在答对或答错时播放提示音。

### 7. 动画反馈

答对时小车会冲刺，答错时小车会抖动，让练习过程更有游戏感。

## 如何运行

### 方法一：直接打开

1. 下载或克隆本项目到本地
2. 找到 `index.html`
3. 用浏览器打开即可开始游戏

### 方法二：使用本地服务器运行

如果你希望通过本地服务访问，可以在项目目录中执行：

```bash
python -m http.server 8000
```

然后在浏览器中打开：

```text
http://localhost:8000
```

## 使用说明

1. 打开页面后，点击 **开始游戏**
2. 查看页面中显示的目标字母
3. 在键盘上按下对应字母
4. 按对后系统会进入下一个字母
5. 按错后系统会提示正确目标字母
6. 在倒计时结束前尽可能获得更高分数

也可以按 **Enter** 键开始游戏。

## 游戏规则

- 每次按对目标字母：加分
- 连续答对：增加连击，并获得额外分数
- 按错字母：扣 3 分，最低不会低于 0 分
- 游戏结束后：显示最终得分和最高连击

## 适合场景

- 儿童键盘启蒙
- 英文字母识别练习
- 打字前的键位熟悉训练
- 亲子互动小游戏
- 小学信息课、英语课辅助练习

## 浏览器兼容性

建议使用最新版浏览器：

- Chrome
- Edge
- Safari
- Firefox

声音朗读功能依赖浏览器的 `speechSynthesis` 能力，不同浏览器的语音效果可能会略有差异。
